Detailed Description

Ouvre une simple fenêtre Windows et dessine un cube avec un blending grace à un fichier BITMAP avec une rotation obtenue par les flêches du clavier.

Modified smoothly by Thierry DECHAIZE

Paradigm : obtain one source (only one !) compatible for multiple free C Compilers and provide for all users an development environment on Windows (64 bits compatible), the great Code::Blocks manager (version 20.03), and don't use glaux.lib or glaux.dll. 

a) Mingw 32 and 64 bits : downloadable http://sourceforge.net/projects/mingw/, but also on https://winlibs.com/

b) Mingw 64 included included in new IDE Red Panda Dev-Cpp, donwloadable http://sourceforge.net/projects/dev-cpp-2020/ c) Mingw 64 bits included in CB : included in package Code::Blocks (version 20.03) http://sourceforge.net/projects/codeblocks/files/Binaries/20.03/Windows/ d) Borland C/C++ 32 bits : downloadable http://edn.embarcadero.com/article/20633 e) Digital Mars Compiler C/C++ version 8.57 32 bits, downloadable http://www.digitalmars.com f) OpenWatcom 32 et 64 bits, version 2.0, downloadable http://openwatcom.mirror.fr/ g) Lcc and Lcc64, 32 et 64 bits : downloadable http://www.cs.virginia.edu/~lcc-win32/ h) Cygwin64 32 et 64 bits : downloadable http://www.cygwin.com/install.html (setup_x64.exe) i) Visual Studio 2022, 32 et 64 bits: community edition for free http://www.microsoft.com/france/visual-studio/essayez/express.aspx j) TDM GCC 32 et 64 bits version 9.2.0, downloadable on http://sourceforge.net/projects/TDM-GCC k) PELLES C 32 et 64 bits version 11.0, downloadable on http://www.smorgasbordet.com/pellesc/ l) MSYS2 environnement MINGW64 : version de 2020 (msys2-x86_64-20201109.exe) on https://repo.msys2.org/distrib/x86_64/msys2-x86_64-20201109.exe m) CLANG, adossé aux environnements MINGW64 et MINGW32, version 13.0.0 (version gcc 11.2.0) downloadable on https://winlibs.com/ n) CLANG, adossé aux environnements Visual Studio 2022 (+ kits Microsoft), version 13.0.0 downloadable on https://releases.llvm.org/download.html
